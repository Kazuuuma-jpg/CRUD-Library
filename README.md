#CRUD-Library

A simple CRUD application to manage a library system, built with PHP, MySQL, and JavaScript.

#How to Download the Files
 - Extract the ZIP file.
 - Move the extracted folder into the htdocs directory.
 - Uploading the Database

#Run XAMPP or Laragon:
 - Open phpMyAdmin through http://localhost/phpmyadmin.
 - Create a new database, then import the SQL file located inside the CRUD folder.

#Running the UI:
 - Open a new browser tab and enter the URL: http://localhost/CRUDPerpustakaan/
 - Once the browser opens the project, you can start navigating through the library system UI.
